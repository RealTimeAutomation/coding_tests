For each bug, document:
- **Module:** Which file/function
- **Severity:** High/Medium/Low
- **Description:** What the bug is
- **Reproduction:** How to trigger it (test case)
- **Impact:** What happens when it occurs
- **Status:** Found / Fixed / Deferred
